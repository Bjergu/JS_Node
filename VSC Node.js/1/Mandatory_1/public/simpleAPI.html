Getting started

In project folder create file called "package.json". This is the folder where all dependecies, scripts etc. will be found.

We will need express framework for this projekt. Therfore we need to make a dependency for express. In package.json make an JSON object wtih "dependecies" as key and another JSON object as value. In the secont JSON object enter "express" as key and the newest verion as value.(should be auto-entered after typing "express")

Now it is time to install express.

Open terminal and make sure to enter projekt folder. Then enter "npm install". After finnish, console should look like this.

Make sure that packages was installed succesfully by looking at project folder. If two new follders like on the pictures apeared, the installation was succesfull.
