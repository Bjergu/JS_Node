Git and Terminal

Git commands

git clone "URL" - Clone git repository and saves it on your desktop.

git add . - Adds changes in the working directory to the ataging area. It tells Git that you want to include updates to a particular file in the next commit. You can write "git add (file)"" or "git add (.)" to include all changes.

git commit -m "first commit" - Commits let you make changelog of incoming and old pushes.

git push -u origin main - execute all added changes included in commit.

Terminal commands

npm install - installing modules included in package.json in project directory.

npm install express - installing express framework.

npm install -g nodemon - installing nodemon globally for all projects on your pc.

npm install --save-dev nodemon - installing nodemon for current projekt.

nodemon file - runs nodemon.
